SUBIR UM PROJETO EXISTENTE PARA O GITHUB

* Criar um repositório no GitHub
* Ir até o Git Bash
* git init (ir até o drive no pc onde está o arquivo pra subir no github)
* git remote add origin https://github.com/marceloShibuya/<nomeprojeto>
* git commit -m "primeiro commit
* git push origin master
* se der o erro [rejected] master -> master(fetch first) ....
* git pull https://github.com/marceloShibuya/<nomeprojeto>
* git fetch origin master (talvez não seja necessário este comando)
* git merge origin master (talvez não seja necessário este comando)
* git fetch origin master:tmp
* git rebase tmp
* git push origin HEAD:master
* git branch -D tmp
* git push origin master
